Σήμερα, δεν είναι πια έτσι: οι σελίδες phishing έχουν γίνει αδιάκριτες με γυμνό μάτι.

## ✨ Τι κάνει το ShieldSign για εσάς

### ✔️ Αναγνωρίζει αυτόματα σελίδες σύνδεσης

Μόλις μια ιστοσελίδα περιέχει πεδίο κωδικού, το ShieldSign ελέγχει τον τομέα.

### ✔️ Εμφανίζει ένα **αδύνατο να μιμηθεί** πανό επικύρωσης

3 λειτουργίες ασφαλείας:

* **Διακριτικό σήμα**
* **Προσωπική λέξη-κλειδί** που γνωρίζετε μόνο εσείς
* **Τυχαίος κωδικός που αλλάζει σε κάθε επίσκεψη** (συνιστάται)

Ένας επιτιθέμενος **δεν μπορεί να αναπαράγει** αυτόν τον κωδικό ή τη λέξη-κλειδί.
Ακόμη και ένα τέλειο αντίγραφο της σελίδας θα αποκαλυφθεί σε ένα δευτερόλεπτο.

### ✔️ Λειτουργεί με λίστες: κοινότητα, προσωπική, επιχείρηση

* **Λίστα κοινότητας** αξιόπιστων τομέων
* Τους **δικούς σας αγαπημένους τομείς**
* Επιλογή επιχείρησης: **λίστα που προωθείται από τον οργανισμό σας**, για ασφάλεια εσωτερικής πρόσβασης

### ✔️ Προσθέτει αυτόματα τις νόμιμες ιστοσελίδες σας

Κάθε φορά που συνδέεστε σε μια νέα ιστοσελίδα, το ShieldSign μπορεί να προσφέρει να την αποθηκεύσει.
**Δεν χρειάζεται να διαχειριστείτε τίποτα: η επέκταση μαθαίνει μαζί σας.**

### ✔️ 100% τοπικό, 100% ιδιωτικό

* Δεν αποστέλλονται δεδομένα σε διακομιστή.
* Χωρίς παρακολούθηση.
* Η λίστα ιστοσελίδων που χρησιμοποιείτε παραμένει **στο πρόγραμμα περιήγησής σας**, μόνο ως URLs.

Επειδή το "κοίταγμα στο URL" δεν είναι πια ρεαλιστικό: κουρασμένος, βιαστικός, αγχωμένος... οι άνθρωποι εξαπατώνται.

Το ShieldSign σας δίνει **έναν εξωτερικό οπτικό δείκτη**, ασφαλή, σταθερό, αξιόπιστο.

Δεν αναρωτιέστε πια:

> "Είναι αυτή η σωστή ιστοσελίδα;"
> Απλά κοιτάτε:
> **Υπάρχει το πανό; Τότε είναι καλό.**
