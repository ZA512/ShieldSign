## ✨ ShieldSign 为您做什么

### ✔️ 自动识别登录页面

一旦网站包含密码字段，ShieldSign 就会检查域名。

### ✔️ 显示**无法模仿的**验证横幅

3 种安全模式：

* **低调徽章**
* 只有您知道的**个人关键词**
* **每次访问都会更改的随机代码**（推荐）

攻击者**无法重现**此代码或关键词。
即使是页面的完美副本也会在一秒钟内被揭穿。

### ✔️ 适用于列表：社区、个人、企业

* 可信域名的**社区列表**
* 您自己的**收藏域名**
* 企业选项：由您的组织推送的**列表**，以保护内部访问

### ✔️ 自动添加您的合法网站

每次登录新网站时，ShieldSign 都可以提供保存它。
**您无需管理任何内容：扩展程序与您一起学习。**

### ✔️ 100% 本地，100% 私密

* 不向服务器发送数据。
* 无跟踪。
* 您使用的网站列表**保留在您的浏览器中**，仅作为 URL。
