## 🛡️ ShieldSign — **パスワードを入力できる場所を知る**最も簡単な方法

ログインページが本物か偽物かを知ることは明白であるべきです。
今日ではもはやそうではありません。フィッシングページは肉眼では区別がつかなくなっています。

**ShieldSignはこの問題を解決します。**
この拡張機能は安全なログインページを自動的に検証し、模倣不可能な明確で見やすいバナーを表示します。
**ShieldSignがページを確認しない場合は、パスワードを入力しないでください。**

## 🎯 なぜこれが本当に便利なのか？

今日、私たちは*不注意によってではなく*騙されています。偽サイトが本物と**完全に同一**である可能性があるからです。
唯一の重要な質問は次のようになります。

👉 *「ここにパスワードを入力できますか？」*

ShieldSignを使用すると、答えは**視覚的で即座に信頼できるもの**になります。

## ✨ ShieldSignがあなたのためにすること

### ✔️ ログインページを自動的に識別

サイトにパスワードフィールドが含まれるとすぐに、ShieldSignはドメインをチェックします。

### ✔️ **模倣不可能な**検証バナーを表示

3つのセキュリティモード：

* **目立たないバッジ**
* あなただけが知っている**個人的なキーワード**
* **訪問ごとに変わるランダムコード**（推奨）

攻撃者はこのコードやキーワードを**再現できません**。
ページの完璧なコピーでさえ、一秒で見破られます。

### ✔️ リストで機能：コミュニティ、個人、エンタープライズ

* 信頼できるドメインの**コミュニティリスト**
* あなた自身の**お気に入りドメイン**
* エンタープライズオプション：**組織によってプッシュされたリスト**で、内部アクセスを保護

### ✔️ 正規のサイトを自動的に追加

新しいサイトにログインするたびに、ShieldSignは保存を提案できます。
**何も管理する必要はありません。拡張機能はあなたと一緒に学習します。**

### ✔️ 100%ローカル、100%プライベート

* サーバーに送信されるデータはありません。
* トラッキングなし。
* 使用するサイトのリストは**ブラウザ内**にURLとしてのみ保存されます。
